My Skills

HTML

HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk membuat dan menyusun halaman web. HTML mendefinisikan struktur dasar sebuah halaman web dengan cara menyusun elemen-elemen yang bisa dilihat dan berinteraksi di browser. HTML terdiri dari berbagai elemen yang berbentuk tag dan atribut, yang memungkinkan pengembang web untuk menampilkan teks, gambar, video, tautan, form, dan berbagai konten lainnya di situs web..

CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan halaman web yang ditulis dengan HTML. CSS memungkinkan pengembang web untuk mengubah elemen-elemen HTML agar terlihat lebih menarik, mengatur layout, warna, font, dan banyak aspek visual lainnya.
